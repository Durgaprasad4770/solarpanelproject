Solar Funding Project

The Solar Funding Project is a user-friendly web application designed to help individuals easily understand and apply for government subsidies related to the installation of solar power systems.

Subsidy Eligibility Calculator: Users can input key data (e.g., system size, location) to instantly calculate potential subsidy amounts.

Dynamic UI: The application uses JavaScript to provide a dynamic and responsive user experience, displaying results and form updates in real-time.

Simple Interface: A clean, accessible interface built with HTML for easy navigation and data entry.

Information Hub: Provides essential links and summarized information about current solar funding schemes.

Technology Stack
Frontend: HTML5, CSS3

Logic: Vanilla JavaScript
